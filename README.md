# JLUcs-StudentScoreSystem-C
程设的第一个项目，自存。用C语言实现本科生成绩管理系统，C++实现页面的图形化。时间有限，当时还有一部分没做完（教务端），设计方面后来也意识到有问题，因此搁置。  

除了普通对链表的增删改查，还有以下一些创新的地方
亮点：  
1.学生端和教师端多次输入密码错误会导致账号冻结，需要教务端解冻  
2.初次使用初始密码登录时会有提示新密码  
3.忘记密码还可通过身份证号找回密码  
4.教务端首次登录时，需要设置密码，后续再次进入教务端再次登录  
5.加入了运行时同时播放音乐的功能  

不足：因为是刚接触cs时做出来的项目，页面都比较朴素，内部逻辑也有点问题。
